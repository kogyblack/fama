## **FAMa** architecture

The project subdivides into four main classes:

- Corpus
  Deals with the datasets.

- Avaliador
  Evaluates performance of a classifier applied to a dataset.

- Treinador
  Machine learning trainer.

- Classificador
  Machine learning classifier
